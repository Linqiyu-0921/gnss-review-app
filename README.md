# 🛰️ GNSS 复习大冒险

仿多邻国风格的 GNSS 考试复习应用，覆盖 8 章 PPT + 名词解释 + 18 条重点共 380+ 道题。

## ✨ 特色

- **多邻国风 UI**：❤️ 心数 / 🔥 连续天数 / ⚡ XP / 💎 钻石
- **学习路径**：8 个单元（对应 PPT 8 章）→ 每单元 4-5 个关卡节点 → 🏆 皇冠终点
- **三种题型**：
  - 🃏 闪卡（翻转 + 4 档判定）
  - 🎯 选择题（4 选项 + 解释）
  - ✏️ 填空题（智能匹配）
- **奖励系统**：XP 经验 / 钻石 / 连击 / 正确率
- **个人页**：成就墙（8 枚勋章）/ 章节进度条 / PPT 网格 / 错题本 / 重置
- **数据持久化**：localStorage 自动保存进度

## 📚 数据来源

- `复习.docx`（8 章考点清单）
- `题目.doc`（8 套带答案模拟试题）
- 18 条期末重点（手写版）
- 老师划重点录音
- 8 章 PPT（图片型 PDF，作为参考材料）

## 🚀 本地运行

由于浏览器 `file://` 协议安全限制，需用 HTTP 服务器：

```bash
# Python 3
python3 -m http.server 8731

# Node.js
npx http-server -p 8731
```

然后访问 http://localhost:8731

4. **错题本**反复回看
5. **PPT 参考**配合翻看原书页
